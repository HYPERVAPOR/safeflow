## 📐 现代前端设计指令：深色主题规范

**🌙 设计风格**: 采用现代深色主题 (Dark Mode)，提供沉浸式的专业用户体验，减少眼部疲劳，适合长时间使用。

### 一、配色方案 (Dark Color Palette)

此方案基于深色背景和柔和的专业强调色，营造现代、专业的深色视觉效果。

| 名称 | 颜色值 (HEX) | 用途描述 |
| :--- | :--- | :--- |
| **背景色 (Primary BG)** | `#0F172A` | 主界面和内容区域的深色背景。 |
| **深背景色 (Secondary BG)** | `#1E293B` | 用于侧边栏、卡片容器或分割区域的深色背景。 |
| **卡片背景 (Card BG)** | `#334155` | 卡片、弹出框、输入框的背景色。 |
| **边框/分割线 (Border)** | `#475569` | 分割线、输入框边框、非悬停状态下的卡片边框。 |
| **强调色 (Accent)** | `#14B8A6` (现代青绿色) | 主要操作按钮 (Primary Button)、选中的 Tab、高亮文本、链接。 |
| **文本-主 (Text-Primary)** | `#F1F5F9` | 主要正文、标题、最重要的信息。 |
| **文本-中 (Text-Secondary)** | `#CBD5E1` | 次要信息、描述文本、提示信息。 |
| **文本-浅 (Text-Tertiary)** | `#94A3B8` | 辅助文本、版权声明、图标默认色。 |
| **成功色 (Success)** | `#22C55E` | 成功通知、状态指示。 |
| **警告色 (Warning)** | `#F59E0B` | 警告信息、需要注意的状态。 |
| **错误色 (Error)** | `#EF4444` | 错误信息、删除操作按钮。 |
| **悬停状态 (Hover)** | `#475569` | 按钮、卡片、列表项的悬停背景色。 |

### 二、排版规范 (Typography)

我们采用一个基于 **Inter** (或类似的系统字体，如 PingFang SC/Microsoft YaHei) 的清晰字体栈。

#### A. 字体家族 (Font Family)

* **UI/正文：** `Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans", sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji"`
* **代码/数据：** `SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`

#### B. 字号与层级 (Font Sizes)

使用 **Rem** 单位以保持缩放友好。基础字号定为 $16px (1rem)$。

| 元素 | 字号 (px) | 字重 (Weight) | 颜色 | 用途 |
| :--- | :--- | :--- | :--- | :--- |
| **H1/主标题** | 30px (1.875rem) | Bold (700) | `#F1F5F9` | 页面标题。 |
| **H2/副标题** | 20px (1.25rem) | Semi-Bold (600) | `#F1F5F9` | 区块标题、大型卡片标题。 |
| **正文/标准** | 16px (1rem) | Regular (400) | `#F1F5F9` | 主内容、表单标签。 |
| **次要文本** | 14px (0.875rem) | Regular (400) | `#CBD5E1` | 表格内容、导航项、描述性文字。 |
| **小型/提示** | 12px (0.75rem) | Regular (400) | `#94A3B8` | 提示文本、元数据、脚注。 |

### 三、间距与尺寸 (Spacing & Sizing)

采用**4px/8px 网格系统**，所有内外边距、组件高度和圆角应是 4 的倍数，以确保整体的视觉韵律和一致性。

| 名称 | 值 (px) | 用途描述 |
| :--- | :--- | :--- |
| **基础边距 (S)** | 4px | 元素间最小间隙，图标与文本间距。 |
| **标准边距 (M)** | 8px | 按钮/输入框的垂直内边距、列表项间距。 |
| **中等边距 (L)** | 16px | 卡片内边距、水平列表项间距、侧边栏元素间距。 |
| **大型边距 (XL)** | 24px | 页面区块间距、模态框内边距。 |
| **卡片圆角 (Border Radius)** | 6px | 按钮、输入框、卡片和模态框的标准圆角。 |
| **组件高度 (Input/Button)** | 36px 或 40px | 标准按钮和输入框的高度。 |
| **图标尺寸 (Icon Size)** | 18px 或 20px | 行内图标的标准尺寸。 |
