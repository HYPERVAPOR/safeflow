# SafeFlow - 基于 LLM Agent 的智能测试平台接入系统

## 项目概述

SafeFlow 是一个根据不同测试任务需求、可进行任意工具嵌入的技术和框架，提供一个智能、高效、灵活、实用的软件测试平台嵌入集成智能一体化系统。通过 LLM Agent 技术和 MCP (Model Context Protocol) 协议，实现外部测试工具的无缝接入和自动适配。
项目prd参考 /docs/prd.md
前端UI风格参考 /docs/how-to-design.md


## 注意事项
前端是nextjs，后端是fastAPI，使用uv做包管理器。
用户已经在本地启动了支持热重载的前后端服务，前端位于3000，后端位于8000，你无需再次启动。
